## 概述

iMin 设备内置热敏打印机，可通过 SDK 直接打印小票。所有 iMin 产品系列均配备集成打印功能。

### 设备分类

iMin 产品分为三大类别：
- **手持金融系列**：便携式移动支付和小票打印设备
- **平板终端系列**：适用于零售和餐饮的平板式终端
- **台式收银机设备**：适用于大批量操作的全功能 POS 终端

### 打印机规格

iMin 设备配备两种类型的内置热敏打印机：
- **80mm 打印机**：纸张宽度 80mm，带切刀功能，兼容 58mm 纸张
- **58mm 打印机**：纸张宽度 58mm，紧凑设计，不带切刀功能

## SDK 版本兼容性

### SDK 2.0（推荐）
- **Android 版本**：Android 13 及以上
- **功能特性**：完整功能集，包括标签打印、文字图片、缓冲区管理和高级配置
- **文档地址**：
  - 中文：https://iminsoftware.github.io/imin_printer/#/zh-cn/v2/api
  - English: https://iminsoftware.github.io/imin_printer/#/v2/api

### SDK 1.0（旧版）
- **Android 版本**：Android 11 及以下
- **功能特性**：基础打印、图片打印、条形码/二维码支持
- **文档地址**：
  - 中文：https://iminsoftware.github.io/imin_printer/#/zh-cn/api
  - English: https://iminsoftware.github.io/imin_printer/#/api

> **注意**：建议新项目使用 SDK 2.0 以获取最新功能和改进。SDK 1.0 仅为旧设备提供支持。
